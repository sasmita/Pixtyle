TO USE THE APPLICATION ON AN ANDROID DEVICE

 Installing Applications With Android SDK
 ----------------------------------------

1. Download and install the Google Android SDK program and the Android USB drivers
   Here is the following link http://code.google.com/android/intro/installing.html

2. Inorder to allow the installation of application from other sources, maodify Android's settings.
   settings -> Application settings -> enable "Unknown Sources". 
   Also under “Settings,” select “SD Card” and “Phone Storage,” and finally enable “Disable Use for USB Storage”
 
3. Open Command Prompt and type the following: adb install path/filename.apk

4. Now Application is ready to use.
